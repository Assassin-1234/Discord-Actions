# Discord-Actions
Use it to get gif for commands like pat, slap, poke, hug and more!!!!

# Installation
<b><a href = "https://www.npmjs.com/package/discord-actions"> npm: </a><b> 
<p>
<code> npm install Discord-Actions </code>
